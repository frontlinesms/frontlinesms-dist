The Settings Menu

The Settings menu allows you to change screen languages and configure preferences.

Appearence

Language

FrontlineSMS currently comes with language support for English, Arabic, Azerbaijani, Bengali, German, Spanish, Finnish, French, Hindi, Indonesian, Khmer, Portuguese, Russian, Swahili and Chinese. Selecting one of these languages from the Language menu will turn all menus and screen messages from the default English into the chosen language. Languages such as Chinese will only display correctly if your computer supports the additional character sets required. If the typeface displays as square blocks on your screen then your computer does not have the correct font. Refer to your Windows user manual for details.

Preferences

From version 1.7.0, FrontlineSMS enables you to access Appearence, General, Database Settings, E-mail Settings, Devices, Internet Services, MMS, HTTP Trigger and Sync from the Preferences Tab.

Home Tab Logo Settings

If you want to customise the "Home" screen in FrontlineSMS to display your own logo or organisation name, you can replace the default "\o/" image with one of your own, or set it as invisible. Because of the smaller screen sizes on some of the lower-cost netbooks and laptops, you may need to remove the image altogether in order to see the whole screen.

General

Statistics

FrontlineSMS collects information on the version installed, User Id, OS, Date of last submision, Contacts created, Number of messages sent and received, Number of messages sent and received since last submission, number of Keywords created, Keyword Actions, Number of Phones connected and Phone details. This information helps us report to our donors and gives a snapshot of how FrontlineSMS is being used. We promise not to use it for anything else.

Cost estimator

FrontlineSMS enables you to set the standard charges per message in the cost estimator section.

Country

When creating a contact, FrontlineSMS can automatically convert the contact number into an international format using the set country's code. Set the country in this section in order to ensure the correct international code is used.

Database Settings

Only under very unusual circumstances should you need to go into the Database Settings menu (for example, to rename the database if you're running two copies of FrontlineSMS on the same machine). It is recommended that only system administrators make any changes to the database.

E-mail Settings

FrontlineSMS is able to turn incoming text messages into emails (see the 'Email' Keyword Action on the Keywords help page). In order for FrontlineSMS to do this, it needs access to a registered email account. This is set up in the Email Settings screen (you may have to refer to your email provider to obtain the necessary settings).

Services

Devices

Under the devices > general section there is a a checkbox labeled 'prompt a help dialog when no device is connected' which is checked by default. The dialog below will be displayed when the application is started and no connected device is detected or whenever a detected device is disconnected.
When a connected device is selected from the device(s) list, the screen below will be displayed to enable the user to set different actions on the connected device.

Internet Service Settings

Although FrontlineSMS was originally developed to allow group text messaging in places where there wasn't internet access, more recent versions allow messages to be sent through the internet if and when it is available. Messages sent this way are generally much faster and cheaper than sending through an attached mobile phone or modem, but you do need to set up an account and buy messages up-front using a credit card. Check the Internet Services help page to learn more about Internet Services in FrontlineSMS.

MMS

From version 1.6.15, FrontlineSMS comes with a new Multimedia Messages (MMS) compatibility. For now, you can receive MMS sent to an e-mail address (MMS sent to the connected device is not supported yet). To do so, just configure your e-mail accounts, and the fetching frequency (see below), and FrontlineSMS will look for MMS in your inbox.

Read more about MMS on the Multimedia Messages help page.

Every carrier formats MMS email content differently, so although MMS from most networks should work without trouble, some may not. If you are finding messages are not processed, or contain incorrect information, please forward the problematic emails to us and we will endeavour to add support in the next release.

FrontlineSMS Plugins Section

All the plugins added to frontlineSMS are listed under this sections. This enables the users to easily access the plugins and manage their settings. HTTP Trigger and Sync are loaded to this section by default.
